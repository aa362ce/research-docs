# FeathersJS

## Before you begin

Install feathersJS cli

```bash
$ npm install @feathersjs/cli -g
```

{% hint style="info" %}
 FeathersJS requires node version 10 or later at the time of writing this latest version for feathersJS is 4.1.0
{% endhint %}
